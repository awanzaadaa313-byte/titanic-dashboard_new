# Titanic Data Visualization Dashboard

This is a Streamlit-based data visualization application designed to analyze the classic Titanic dataset.

## Features
- 3 Dynamic KPI Cards (Total Records, Average Age, Average Fare)
- Sidebar dropdown filters for Gender and Ticket Class
- 10 Unique data analysis charts built with Matplotlib and Seaborn

## How to Run the Project
1. Install dependencies: `pip install -r requirements.txt`
2. Launch the dashboard: `python -m streamlit run app.py`
